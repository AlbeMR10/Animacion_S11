# Animacion_S11

Este proyecto implementa una escena 3D interactiva utilizando Three.js para el renderizado y Ammo.js para la simulación física.

El proyecto recrea una pista de bolos en la que el jugador puede moverse libremente en primera persona y lanzar una bola física que colisiona con los bolos y con una diana explosiva, la cual se fragmenta en varios trozos al ser golpeada.

Para moverse por el entorno:

W : Adelante 
A : Izquierda    
S : Atrás 
D : Derecha

Espacio : Lanzar la bola

Apuntar: Movimiento del ratón
